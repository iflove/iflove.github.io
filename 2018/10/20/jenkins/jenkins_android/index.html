Jenkins上构建Android指南

Jenkins 是款牛逼的自动化服务器 ,提供n个插件来支持构建，部署和自动化任何项目。因为项目需要，特来搞搞

文档直通车 看文档操作再简单不过了
cd /Applications/Jenkins
java -jar jenkins.war --httpPort=8080
打开浏览器输入 http://localhost:8080

要在Jenkins上构建Android，Android 开发环境必不可少的

可以点击系统管理-系统设置 配置好环境变量

可以点击系统管理-全局工具配置
配置好JDK、Gradle

点击 配置 进入项目构建详细配置
Android 多渠道版本构建，可以使用参数化构建过程 -添加选项参数

源码管理可以根据自己项目来配置

构建步骤选Invoke Gradle script
关于Tasks，可以运行 gradle tasks 查看现在能用的tasks

总体配置一个简单的Android 构建任务还是比较简单的。如果遇到问题，欢迎交流
